## My Process

1. **Read the problem** — understand inputs, outputs, constraints
2. **Try for 20 minutes** — genuine effort before looking anything up
3. **If stuck** — watch NeetCode explanation, then close it and write solution from scratch
4. **Fill in the file header** — approach, complexity, gotchas (can't skip this)
5. **Rate comfort level** — 1–3 means revisit in 2 weeks

## Pattern Cheat Sheet

| Pattern | Use When | Classic Example |
|---------|----------|-----------------|
| Arrays & Hashing | Need O(1) lookup, frequency count | Two Sum |
| Two Pointers | Sorted array, pairs, palindromes | 3Sum |
| Sliding Window | Contiguous subarray with constraint | Longest Substring |
| Stack | Matching brackets, next greater element | Valid Parentheses |
| Binary Search | Sorted input or search space reduction | Search Rotated Array |
| Linked List | In-place reversal, cycle detection | Reverse Linked List |
| Trees (DFS) | Path problems, structure validation | Validate BST |
| Trees (BFS) | Level-order, shortest path | Level Order Traversal |
| Heap | K-th largest/smallest, streaming data | Kth Largest Element |
| Graphs | Connected components, shortest path | Number of Islands |
| Dynamic Programming | Overlapping subproblems, optimal choices | Coin Change |
| Backtracking | Generate all combinations/permutations | Subsets |
| Greedy | Local optimal → global optimal | Jump Game |
